# Project Name: Tsehafi
## Built With
- Languages: Ruby
- Framework: Ruby on Rails

## Getting started
To get a local copy of this exercise, Please follow these simple example steps.

```bash command
$ git clone https://github.com/IamYunusAli/Tsehafi.git
$ cd tsehafi
$ bundle install
$ bin/rails db:create
$ rake db:migrate
$ rails s
```
